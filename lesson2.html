Lesson 2 · 형용사로 묘사하기 Describing things you like

학습 목표 Goals

좋아하는 것을 한국어로 묘사해 봐요 — 음식, 게임, 취미처럼 가족과 이야기할 만한 것들로요. 형용사를 자연스럽게 활용하고, 1과에서 익힌 조사·연결을 함께 써서 문장을 만들어요. (This lesson is about describing things you like, using adjectives naturally.)

✓ 형용사를 해요체로 바꾸기 (맵다 → 매워요) Conjugate adjectives
✓ N은/는 [형용사] 묘사 문장 만들기 Build “N is [adjective]” sentences
✓ 좋아하는 것을 2~3문장으로 소개하기 (말하기 중심) Describe something in 2–3 sentences (speaking)

1. 원리 How it works (묘사)

① 무엇이 어떻다: 「N은/는 [형용사].」 받침 있으면 은, 없으면 는.
음식은 맛있어요 · 노래는 좋아요

② 형용사는 -다를 떼고 -아/어요: 싸다 → 싸요, 바쁘다 → 바빠요 (불규칙도 있어요!)

③ 명사 꾸미기(관형형): [형용사]+ㄴ/는 + 명사 → 맛있는 음식, 재미있는 책

2. 핵심 어휘 Adjectives · 묘사 형용사

아래 형용사로 다음 연습을 해요. (활용형은 3번에서 직접 만들어 봐요.)

형용사뜻형용사뜻

3. 해요체로 바꾸기 Conjugate · 입력 즉시 채점

형용사를 해요체로 바꿔 쓰세요 (예: 싸다 → 싸요). 불규칙에 주의!

4. 은/는 넣기 Fill the particle

빈칸에 은 또는 는만 넣으세요 (받침 규칙).

5. 묘사 문장 만들기 Make a description

주어진 명사 + 형용사로 「N은/는 [형용사]-아/어요.」 해요체로 쓰세요 (예: 가방은 커요.). 은/는은 초록/빨강으로 채점하고, 형용사가 해요체 정답 형태면 ✓, 아니면 노란색 “해요체 확인”으로 알려줘요 (철자·불규칙 점검은 선생님과).

6. 내가 좋아하는 것 Describe something you like

좋아하는 것 하나를 골라 2~3문장으로 묘사해 보세요 (음식·게임·원예·악기 등). 형용사 + 은/는 + 1과의 연결(-고, -아서)을 써 보세요. — 자동 채점은 없어요. 선생님과 함께 다듬어요.

예: 클라리넷은 어려워요. 그렇지만 정말 멋있어서 저는 클라리넷을 좋아해요.

👩‍🏫 선생님 카드 말하기·듣기 + 과제

괄호는 학습자가 떠올릴 만한 형용사 예시일 뿐, 다른 형용사도 환영.
화제: 영화(재미있다) · 음악(좋다) · 운동(어렵다) · 지역 행사(재미있다) · 경제(어렵다) · 음식(맛있다)
일상: 김치(맵다) · 커피(달다) · 날씨(좋다) · 꽃(예쁘다)

① 묘사 말하기 드릴 — 빠른 즉답 (3분)

통제된 자동화 연습. 단어를 빠르게 던지면 학습자가 "N은/는 [형용사]-아/어요" 한 문장으로 즉답해요. 목표는 속도 + 정확성 — 리듬 타며 많이 반복. (이유·확장은 ②에서 다뤄요.)

  • 영화 → "영화는 재미있어요" / 음식 → "음식은 맛있어요" / 꽃 → "꽃은 예뻐요"

② 듣고 길게 말하기 — 대화 끌어내기 (듣기 + 말하기)

선생님은 짧게, 학습자가 길게. 선생님의 역할은 짧은 질문·맞장구로 학습자가 계속 말하게 만드는 거예요. 목표 발화 비중은 학습자 80% · 선생님 20%. 단답으로 끝나면 바로 다음 추임새로 한 번 더 끌어내고, 침묵을 선생님이 채우지 마세요(학습자가 채우게 기다리기).

① 관심 주제로 짧게 운을 떼면, 학습자가 듣고 「형용사 + 은/는 + 연결어미(-고·-아서·-지만)」로 3문장 이상 대답해요. ② 멈추면 아래 짧은 말로 한 번 더 이어가게 해요.

여는 질문 (하나 골라 말로 던지기 — 학습자 관심사):

  • "요즘 무슨 영화 봤어요? 어땠어요?"
  • "요즘 무슨 음악 들어요? 어때요?"
  • "어떤 운동이 좋아요?" / "동네에 무슨 행사 있었어요?"
  • "요즘 무슨 음식이 맛있어요?"

이어가게 하는 짧은 말 (선생님은 이것만, 내용은 학습자가 채움):

  • "왜요?" · "그래서요?" · "또요?" · "어떻게요?" · "예를 들면요?" · "그다음은요?" · "더 말해 주세요."

듣기 포인트(강점 영역 유지): 중간에 선생님이 새 정보를 한 문장 던지고("저는 매운 음식을 잘 못 먹어요.") 학습자가 듣고 반응·비교하게 해요(예: "그래요? 저는 매워서 더 좋아해요"). 듣고 즉시 받아치는 연습.

성공 기준: 한 주제로 학습자가 5문장 이상 이어서 말하고, 학습자 발화가 선생님의 3~4배가 되면 잘된 거예요.

③ 과제 Homework

  • 가족에게 소개하고 싶은 "내가 좋아하는 것" 하나를 3~4문장으로 묘사 (음식·게임·원예·클라리넷 등). 형용사 + 은/는 + 연결어미 사용.

평가 포인트: ① 형용사 활용이 맞는가(매워요·예뻐요 등) ② 은/는을 빠뜨리지 않는가 ③ 문장을 연결하는가. 말하기 유창성도 함께 관찰.
