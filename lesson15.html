0. 미니 복습 Quick review

지난 과들에서 짧게. 답은 모두 해요체로 쓰세요 (기본동사는 [ ] 안에).

1. 원리 How it works

① 형용사 + -(으)ㄴ 거 = "a ~ one" — 물건을 고를 때 핵심!
크다 → 큰 거 · 싸다 → 싼 거 · 다르다 → 다른 거 · 작다 → 작은 거
「더 ~ㄴ 거 있어요?」 = Do you have a ~er one? 받침 없음 → -ㄴ(큰·싼), 받침 있음 → -은(작은·좋은)

② 이거 / 그거 / 저거 = this / that / that over there. 이걸로(=이것으로) 할게요 = I'll take this one.

③ 쇼핑 회화 틀: 「얼마예요?」 How much? · 「이걸로 할게요.」 I'll take this. · 「깎아 주세요.」 Give me a discount. · 「포장해 주세요.」 Please wrap it.

3. -(으)ㄴ 거 만들기 "a ~ one" · 입력 즉시 채점

형용사를 -(으)ㄴ 거로 바꿔 쓰세요 (예: 좋다 → 좋은 거).

4. -아/어 주세요 만들기 Shop requests

동사를 -아/어 주세요로 바꿔 쓰세요 (예: 찾다 → 찾아 주세요).

5. 🎬 실전 대화 ① — 기념품 고르기 Choosing a souvenir

선생님과 번갈아 소리 내어 읽어요 (손님 ↔ 점원). 둘러보기 → 가격 → 더 큰/다른 거 → 고르기.

🛍️ 기념품점에서 — 둘러보고 고르기
🧳 손님안녕하세요. 그냥 좀 둘러봐도 돼요?
🧑 점원네, 천천히 보세요. 찾으시는 거 있으면 말씀하세요.
🧳 손님이 열쇠고리 얼마예요?
🧑 점원그건 오천 원이에요.
🧳 손님좀 더 큰 거 있어요?
🧑 점원네, 여기 이게 더 커요. 색깔도 여러 가지예요.
🧳 손님그럼 파란색으로 보여 주세요.
🧑 점원여기요. 이거 인기가 많아요.
🧳 손님예쁘네요. 이걸로 할게요.
