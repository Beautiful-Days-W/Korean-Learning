Lesson 7 · 정중하게 부탁·권유하기 Polite requests & offers

학습 목표 Goals

남에게 정중하게 부탁하고 권하는 법을 배워요 — "도와 주세요", "기다려 주시겠어요?"처럼요. (6과는 남의 부탁을 전하기였고, 7과는 내가 직접 정중하게 부탁·권유하기예요.) (How to make polite requests and offers yourself.)

✓ 부탁: -아/어 주세요 Please do ~
✓ 더 정중하게: -아/어 주시겠어요? / -아/어 주실 수 있어요? Would/Could you ~?
✓ 권유·제안: 같이 -(으)ㄹ까요? / -(으)시겠어요? + 격식 어휘(권하다·청하다) Shall we? / Would you like to?

1. 원리 How it works (정중한 부탁)

① 기본 부탁 — V-아/어 주세요: "나를 위해 ~해 주세요" (please do ~ for me). 받침 규칙대로 -아/어 (attach -아/어 by the vowel rule).
돕다 → 도와 주세요 · 읽다 → 읽어 주세요 · 기다리다 → 기다려 주세요

② 더 정중하게(부드러운 질문체) (more polite — soften it into a question):
-아/어 주시겠어요? (Would you ~?) · -아/어 주실 수 있어요? (Could you ~?)
예: 기다려 주시겠어요? / 도와 주실 수 있어요?

③ 권유·제안 (offers & suggestions): 같이 V-(으)ㄹ까요? (Shall we ~?) · V-(으)시겠어요? (Would you like to ~?)
예: 같이 갈까요? · 차 한잔 하시겠어요?

④ 격식 어휘(오늘 배운 것) (formal vocabulary): 권하다 (to offer / recommend) · 청하다 (to request politely) · 식사하다 (polite word for having a meal).

2. 핵심 어휘 Vocabulary · 부탁·권유

표현뜻표현뜻

3. 부탁 만들기 -아/어 주세요 · 입력 즉시 채점

기본형을 「-아/어 주세요」 부탁으로 바꿔 쓰세요 (예: 읽다 → 읽어 주세요). 받침 규칙·불규칙 주의!

4. 더 정중하게 -아/어 주시겠어요?

같은 부탁을 「-아/어 주시겠어요?」로 더 정중하게 바꿔 쓰세요.

5. 정중하게 부탁 쓰기 Write a polite request

누군가에게 정중하게 부탁/권유하는 말을 2문장 써 보세요. (-아/어 주세요 / 주시겠어요? / 같이 -ㄹ까요? 등) — 점수엔 안 들어가고 선생님이 봐요.

예: 선생님, 한 번 더 설명해 주시겠어요? 그리고 천천히 말해 주세요.

👩‍🏫 선생님 카드 말하기·듣기 + 과제

상황·동사를 던지면 학습자가 「…-아/어 주세요 / 주시겠어요?」로 정중히 부탁해요.
부탁 동사: 돕다 · 기다리다 · 읽다 · 열다 · 보내다 · 빌리다 · 설명하다 · 들려주다
권유 상황: 차 한잔 · 같이 식사 · 같이 가기 · 자리 권하기

① 부탁 즉답 드릴 — 빠른 즉답 (3분)

통제된 자동화. 동사를 빠르게 던지면 학습자가 "…-아/어 주세요"로 즉답. 이어서 "주시겠어요?"로 정중도 한 단계 올리기.

  • 돕다 → "도와 주세요" → (정중) "도와 주시겠어요?"
  • 기다리다 → "기다려 주세요" → "기다려 주시겠어요?" / 열다 → "열어 주세요"

② 듣고 길게 말하기 — 부탁·권유 대화 (듣기 + 말하기)

선생님은 짧게, 학습자가 길게. 상황을 주면 학습자가 정중히 부탁/권유하고 이유도 덧붙이게. 발화 비중 학습자 80%.

여는 질문/상황:

  • "길을 모를 때, 어떻게 정중히 물어봐요?"
  • "손님이 왔어요. 뭐라고 권해요?" (예: 차 한잔 하시겠어요?)
  • "가족에게 도움이 필요해요. 정중하게 부탁해 보세요."

듣기 포인트: 선생님이 반말 부탁("이거 해 줘")을 던지면 학습자가 정중체로(해 주세요 / 해 주시겠어요?) 고쳐 말하게.

성공 기준: 정중도(주세요 → 주시겠어요? → 주실 수 있어요?)를 상황에 맞게 올려 5문장 이상.

③ 과제 Homework

  • 정중하게 부탁/권유하는 말 3~4문장 쓰기. -아/어 주세요, -아/어 주시겠어요?, 같이 -(으)ㄹ까요? 중 두 가지 이상 사용. (예: 가족·선생님·이웃에게)

평가 포인트: ① -아/어 주세요 형태가 맞는가(받침·불규칙) ② 더 정중한 형태(주시겠어요?)를 쓰는가 ③ 상황에 맞는 정중도.
