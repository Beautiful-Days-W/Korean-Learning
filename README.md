# Korean lessons (learner site)

A personalized Korean course for an adult English-speaking learner — interactive, self-grading lesson pages.

**Live:** https://korean.songfamily.work
**This repo:** a static mirror of the learner-facing pages, hosted on GitHub Pages.

## Pages
- `index.html` — home
- `숙제.html` — homework submit + feedback
- `평가시험.html` — Lessons 1–5 review test
- `lesson1.html` … `lesson5.html` — 을/를 · 형용사 묘사 · 비교·의견 · 간접화법 · 높임말
- `questionnaire.html`, `assessment.html` — onboarding & placement

> Note: homework/test submission and feedback call the live backend at `korean.songfamily.work`,
> so those features work from this mirror too.
