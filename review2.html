1. 명령·부탁 전하기 Reporting commands/requests 평가 오답

남의 말을 전할 때: 명령은 -(으)라고 하다, "나에게 해 달라"는 -아/어 달라고 하다.
"조용히 해" → 조용히 하라고 해요 · "물 줘"(나에게) → 물을 달라고 해요 (report what someone said, not the command itself).

따옴표 말을 전하는 문장으로 바꿔 빈칸을 채우세요.

2. 권유? 부탁? Suggestion vs request 평가 오답

권유(제안, Shall we?) = 같이 -(으)ㄹ까요? — 우리 함께 해요.
부탁(요청, favor) = -아/어 주세요 / 주시겠어요? — '주다'가 붙으면 "나를 위해 해 줘"라는 부탁이에요! (주다 = a favor → request, not a suggestion.)

괄호의 의도에 맞게 빈칸을 채우세요. 제안은 「같이 ~(으)ㄹ까요?」, 부탁은 「~아/어 주시겠어요?」로 쓰세요.

3. 높임은 사람에게만 Honorifics = people only 계속 나온 실수

-(으)시- 높임은 사람에게만! 사물(가방·아이스크림·음식…)에는 안 써요.
아이스크림이 있으세요? ✗ → 있어요? ✓ / 할머니, 시간 있으세요? ✓ (사람은 OK)

괄호의 동사를 해요체로 쓰세요. 사물이면 높임 없이, 사람이면 -(으)세요로!

👩‍🏫 선생님 카드 말하기 드릴 — 약점 위주

전하기 큐: "조용히 해" · "기다려" · "빨리 와" · (나에게)"물 줘"·"도와줘"
권유/부탁 큐: (제안) 같이 영화·점심·산책 / (부탁) 문 열기·짐 들기·설명하기
사물 높임 함정: 아이스크림·사이즈·가방·김치 (있으세요? → 있어요?)

① 전하기 즉답 (평가 오답 ★)

선생님이 직접 명령을 던지면 학습자가 전하는 말로 바꿔요. "조용히 해" → "조용히 하라고 해요." / (나에게) "물 줘" → "물을 달라고 해요."

② 권유 vs 부탁 골라 말하기 (평가 오답 ★)

상황을 주고 학습자가 맞는 걸 골라 말하게. "같이 영화 보고 싶어 → 제안!" → "같이 볼까요?" / "짐이 무거워 → 부탁!" → "들어 주시겠어요?" 주다 붙으면 부탁, 없으면 권유!

③ 사물 높임 함정 (계속 나온 실수 ★)

가게 상황으로: 선생님이 사물을 대면 학습자가 높임 없이 물어야 함. "아이스크림?" → "있어요?"(○) / "있으세요?"(✗). 사람이면 반대: "할머니?" → "계세요?"(○)

성공 기준: 세 약점(전하기·권유vs부탁·사물높임)을 각각 멈칫 없이 3개 이상.
