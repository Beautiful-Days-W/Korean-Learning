복습 · Lessons 1–5 Review — guessing & honorifics

목표 Goals

평가에서 약했던 두 가지를 짧게 보강해요 — ① 추측 -(으)ㄹ 것 같아요 ② 높임(어휘 + 께서면 서술어도 높이기). (Quick review of two weak spots: the guessing form & honorifics.)

✓ 기본형 → -(으)ㄹ 것 같아요 만들기 Build the guessing form
✓ 주어가 께서면 서술어도 -(으)세요로 + 높임 어휘 Match honorific subject with honorific predicate

1. 원리 How it works

동사·형용사 + -(으)ㄹ 것 같아요 = "~할 것 같다 / ~일 것 같다"(추측).

① 받침 없음 → -ㄹ 것 같아요: 오르다 → 오를 것 같아요 · 가다 → 갈 것 같아요
② 받침 있음 → -을 것 같아요: 좋다 → 좋을 것 같아요 · 먹다 → 먹을 것 같아요
③ ㄹ받침 → 그대로 -ㄹ 것: 살다 → 살 것 같아요 (을 안 붙임)
④ ㅂ불규칙: 어렵다 → 어려울 것 같아요

💡 핵심은 "기본형에서 -다를 떼고 받침 규칙대로 (으)ㄹ 붙이기".

2. 추측 만들기 Make a guess · 입력 즉시 채점

기본형을 「-(으)ㄹ 것 같아요」로 바꿔 쓰세요. 받침 규칙·불규칙에 주의!

3. 짧게 추측 쓰기 Write a short guess

날씨·시장·주말 등 무엇이든, -(으)ㄹ 것 같아요로 2문장 추측해 보세요. — 점수엔 안 들어가고 선생님이 봐요.

예: 내일 비가 올 것 같아요. 그래서 사람이 적을 것 같아요.

4. 높임 어휘 빠른 점검 Honorific words

보통말 → 높임말. 웃어른 이야기할 때 써요.

먹다 → 드시다(드세요) · 자다 → 주무시다(주무세요) · 있다 → 계시다(계세요) · 말하다 → 말씀하시다 · 주다 → 드리다

배고프다 → 시장하다 · 아프다 → 편찮으시다 · 나이 → 연세 · 집 → 댁 · 이름 → 성함
※ 시장하다(hungry)는 시장(market)과 소리만 같고 뜻은 무관!

5. 높임 일치 — 께서면 서술어도 높이기 Match the predicate · 입력 즉시 채점

주어에 께서가 있으면 서술어도 -(으)세요로! 아래 문장을 높임말로 고쳐 쓰세요.

👩‍🏫 선생님 카드 말하기 드릴

추측 즉답 드릴 (빠른 즉답)

기본형을 빠르게 던지면 학습자가 "…-(으)ㄹ 것 같아요"로 즉답. 받침 규칙을 입에 붙이는 게 목표예요.

  • 오르다 → "오를 것 같아요" / 좋다 → "좋을 것 같아요" / 살다 → "살 것 같아요"
  • 이어서: "왜 그렇게 생각해요?" 하고 이유(-기 때문에)까지 말하게

높임 일치 즉답 드릴

"할아버지께서 + 배고프다"처럼 던지면 학습자가 "할아버지께서 배고프세요"로 즉답. 께서면 서술어도 높이는 습관을 들이기.

  • 어머니께서 / 좋아하다 → "어머니께서 좋아하세요" · 선생님께서 / 가다 → "선생님께서 가세요"
