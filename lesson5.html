Lesson 5 · 높임말로 어른께 말하기 Speaking politely to elders

학습 목표 Goals

어른께 공손하게 말해 봐요 — 할아버지·할머니·선생님처럼 높여야 하는 분께요. 가족 어른과 대화할 때 꼭 필요해요. (This lesson is about speaking politely to elders using honorifics.)

✓ 주체 높임 -(으)세요 Honorific -(으)세요
✓ 특수 높임 어휘 (드시다·계시다·연세) Special honorific words
✓ 높임 조사 께서 / 께 Honorific particles (이/가→께서, 에게→께)

1. 원리 How it works (높임말)

① 주체 높임 -(으)시- → 권유·요청 -(으)세요: 받침 없으면 -세요, 있으면 -으세요.
가다 → 가세요 · 읽다 → 읽으세요

② 특수 높임 어휘(아예 단어가 바뀜): 먹다 → 드시다(드세요) · 자다 → 주무시다(주무세요) · 있다 → 계시다(계세요) · 말하다 → 말씀하시다.
명사도: 나이 → 연세 · 집 → 댁 · 이름 → 성함 · (드리는) 주다 → 드리다

③ 높임 조사: 이/가 → 께서, 에게 → 께.
할아버지께서 오세요 · 선생님께 드렸어요

2. 핵심 어휘 Vocabulary · 높임 표현

아래 높임 표현으로 다음 연습을 해요.

높임말뜻(보통말)높임말뜻(보통말)

3. 높임 활용 Make it honorific · 입력 즉시 채점

동사를 「-(으)세요」 높임 형태로 바꿔 쓰세요. 〈 〉에 특수 높임 어휘가 있으면 그걸 써요 (먹다→드세요).

4. 께서 / 께 넣기 Honorific particles

빈칸에 께서(이/가 자리) 또는 께(에게 자리)를 넣으세요.

5. 어른께 문장 만들기 Build an honorific sentence

주어진 어른 + 동사로 「(어른)께서 … -(으)세요.」 높임 문장을 쓰세요. 께서는 초록/빨강, 높임 동사 형태가 맞으면 ✓, 빠지면 노란색 “높임 동사 확인”.

6. 어른께 짧게 쓰기 Write to an elder

할아버지·할머니·선생님 같은 어른께 하는 말을 2~3문장 높임말로 써 보세요. (께서/께, -(으)세요, 특수 높임 어휘를 써요.) — 내용 채점은 없어요(형식만 신호). 선생님과 함께 다듬어요.

예: 할아버지께서 댁에 계세요. 할아버지께 제가 점심을 차려 드릴 거예요.

👩‍🏫 선생님 카드 말하기·듣기 + 과제

"(어른) + (동사)"를 던지면 학습자가 「(어른)께서 …(으)세요」로 말해요.
어른: 할아버지 · 할머니 · 어머니 · 아버지 · 선생님
동사: 오다(오세요) · 읽다(읽으세요) · 먹다(드세요) · 자다(주무세요) · 있다(계세요) · 말하다(말씀하세요)

① 높임 즉답 드릴 — 빠른 즉답 (3분)

통제된 자동화 연습. "(어른) + (동사)"를 빠르게 던지면 학습자가 "(어른)께서 …(으)세요" 한 문장으로 즉답해요. 목표는 속도 + 높임 형태 정확성. (확장은 ②에서.)

  • 할아버지 / 읽다 → "할아버지께서 책을 읽으세요"
  • 어머니 / 먹다 → "어머니께서 진지를 드세요" / 선생님 / 있다 → "선생님께서 교실에 계세요"

② 듣고 길게 말하기 — 높임 대화 (듣기 + 말하기)

선생님은 짧게, 학습자가 길게. 짧은 질문·맞장구로 학습자가 어른에 대해 높임말로 길게 말하게 해요. 목표 발화 비중 학습자 80% · 선생님 20%. 단답이면 추임새로 더 끌어내고, 침묵을 선생님이 채우지 마세요.

질문을 던지면, 학습자가 께서/께 + -(으)세요 + 특수 높임 어휘를 섞어 3문장 이상 말하게 해요.

여는 질문 (하나 골라 말로 던지기):

  • "할아버지께서 요즘 어떻게 지내세요?"
  • "주말에 어른들과 뭐 하셨어요?"
  • "선생님께 뭐라고 말씀드릴 거예요?"
  • "가족 어른 한 분을 높임말로 소개해 주세요."

이어가게 하는 짧은 말 (선생님은 이것만):

  • "그래서요?" · "또요?" · "어떻게요?" · "그다음은요?" · "더 말해 주세요."

듣기 포인트(강점 영역 유지): 선생님이 보통말 문장을 던지고("할아버지가 밥을 먹어요.") 학습자가 높임말로 고쳐 말하게 해요(→ "할아버지께서 진지를 드세요").

성공 기준: 한 어른에 대해 학습자가 5문장 이상 높임말로 이어 말하면 잘된 거예요.

③ 과제 Homework

  • 가족 어른 한 분에 대해(또는 그분께 하고 싶은 말을) 3~4문장 높임말로 쓰기. 께서/께 + -(으)세요 + 특수 높임 어휘를 꼭 사용.

평가 포인트: ① -(으)세요 높임 형태 ② 께서/께 조사 ③ 특수 높임 어휘(드시다·계시다 등)를 맞게 쓰는가. 유창성·발화량도 함께 관찰.
